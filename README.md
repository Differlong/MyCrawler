# MyCrawler
整理我写的爬虫代码

1.[guaziScrawler.py](./guaziScrawler.py)
作用：
爬取[瓜子二手车](www.guazi.com)的汽车外观图片，保存链接和车名到指定文件夹，为之后深度学习训练做准备。




2.[多线程美图录爬虫.py](./多线程美图录爬虫.py)
作用：
多线程爬取[美图录](www.meitulu.com)的美女图片，并且保存到指定的文件夹。（福利，值得收藏！）
